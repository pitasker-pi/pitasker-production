Privacy Policy for PiTasker (Tasker for Pi)

Last Updated: June 29, 2025

1. Information We Collect

  • Pi Wallet Address: For processing payments in Pi cryptocurrency

2. How We Use Data

Data is used exclusively to:

  • Facilitate Pi transactions between task posters and workers

  • Prevent fraud using IP address analysis

By using PiTasker, you acknowledge:

  • All payments are processed through Pi blockchain
  • Transaction histories are immutable and public
